1.Pandas概述

  1. Pandas是Python的一个数据分析包，该工具为解决数据分析任务而创建。
  2. Pandas纳入大量库和标准数据模型，提供高效的操作数据集所需的工具。
  3. Pandas提供大量能使我们快速便捷地处理数据的函数和方法。
  4. Pandas是字典形式，基于NumPy创建，让NumPy为中心的应用变得更加简单。
